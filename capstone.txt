2. Key:
- add a file named .env in the the parent TrackerProjectSite directory.

- Enter the following command in the terminal to generate a secret key:
python -c 'from django.core.management.utils import get_random_secret_key; print(get_random_secret_key())'

-In the .env file created above, insert the key and debug mode in the file:
SECRET_KEY=Key-generated-above
DEBUG=False
